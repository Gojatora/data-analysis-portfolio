# Data Analysis Portolio

Hi! I'm Adrian Paolo Follante, an aspiring data analyst based in the Philippines.
This portfolio showcases my Excel data analysis projects.

## Projects

### 1. Teen Mental Health & Social Media Analysis
- **Tools:** Excel, PivotTables, AVERAGEIFS, Charts
- **Key Finding:** Males show higher social media addiction 
  across all platforms. TikTok has the highest addiction scores.

### 2. E-Commerce Sales Analysis
- **Tools:** Excel, Data Cleaning, PivotTables, Dashboard
- **Key Finding:** Books generate the highest revenue but have 
  a 43.5% return rate.

### 3. Clinic Appointments Analysis
- **Tools:** Excel, Power Query, Nested IFs, Dashboard
- **Key Finding:** Neurology is the highest revenue department. 
  General has the highest follow-up rate at 54%.

### 4. Café Sales Analysis
- **Tools:** Excel, Power Query, Data Cleaning, Dashboard
- **Key Finding:** Juice is most popular but Salad generates 
  the most revenue due to higher pricing.
